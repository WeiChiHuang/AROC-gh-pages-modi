Safe Terminal Regions

For the model predictive control algorithms described in Sec. 2.2, one requires a terminal region 𝒯. In this section we present algorithms to compute terminal regions. These algorithms first synthesize a terminal controller uc(x(t), t), and then compute a terminal region for the controlled system.

The terminal region is defined as a set 𝒯 ⊂ ℝn for which the terminal controller steers all points inside the set in finite time back into the set, while satisfying input and state constraints and considering disturbances acting on the system:

𝒯 = { x | ∃ tf ∈ ℝ+ : ∀ w(·) ∈ 𝒲, ∀ v(·) ∈ 𝒱 : ξ(tf, x, uc(·), w(·), v(·)) ∈ 𝒯,

∀ t ∈ [0, tf], ∀ x̂(t) ∈ ℛuc(t) ⊕ 𝒱 : uc(x̂(t), t) ∈ 𝒰,

∀ t ∈ [0, tf], ∀ w(·) ∈ 𝒲, ∀ v(·) ∈ 𝒱 : ξ(t, x, uc(·), w(·), v(·)) ∈ 𝒳 }
